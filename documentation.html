Tea-Fueled Last.fm Class/Library. This is a simple PHP library that allows you to interact with Last.fm's API.

Getting started

Before you start to use this library, you first need to get an API key from Last.fm. You can get one at Last.fm API site. Take note of the API Key.

Now that you have an API Key, you need to include lastfmclass.php in your PHP page.

include('lastfmclass.php');

After you have the library included, you need to create a new object so you can use the library. To do this, you need to assign lastFM to a variable. When you create this object, you need to pass two options to it, the API Key, and your username.

$lastfm = new lastFM('APIKEY','username');

Now you are ready to start using the Last.fm Library.

Making a basic API Calls

Now that you have the Last.fm Library stored in an object, you can now make API calls.

This library does not have support for all of the API calls. To see the calls that are supported you can check the Wiki. Most calls have the same basic naming convention, categoryMethod, leaving off the get. Check source for exact name for the call you want to make.

Here is the syntax to make a call for the latest listen to tracks.

$lastfm->userRecent();

Since version 0.4, all methods pass back an array, so you need to capture this in a variable.

$tracks = $lastfm->userRecent();

Most methods return a multi-level array (or array within an array) so to go through the data, you need to use a for or foreach statement.

foreach($tracks as $track){
	echo $track->name;
}

Some calls allow for options to be passed. Right now, it doesn't have the greatest implementation, but this will change with the next version.

$track = $lastfm->userRecent(10)

The above code adds a limit parameter, so only 10 tracks are passed back.

$track = $lastfm->userRecent(10, 7)

The above code again, adds a limit and adds a time back parameter declared in days. So this call returns 10 tracks from the past 7 days.

Here is the problem with passing options back like this.

$track = $lastfm->userRecent('', 7)

If you just want to pass back the time parameter, you have to add a blank parameter for the limit. This will be changing in the next version.

If you want to know what options each call has, please check the source.
