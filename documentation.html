Tea-Fueled Last.fm PHP Class

This is a new project, things will change, so I didn't spend much time on this documentation, because things will probably change. But for early adopters, here is how you can use it now.

First you have to call lastfmclass.php by using include().

<?php include("lastfmclass.php"); ?>

Then you initialize it:

<?php $lastfm = new lastFM(); ?>

Then you need to set some parameters.

<?php
	$lastfm->user = "yourusername";
	$lastfm->apikey = "yourApiKey";	
?>

This class requires that you use your own API key. It's simple to get one, just head over to http://www.last.fm/api/account.

So all together, it should look something like:

<?php
	include('lastfmclass.php');
	$lastfm = new lastFM();
	$lastfm->user = "username";
	$lastfm->apikey = "apikey";
?>

Now we can make a Last.fm API call.

Currently there are only a couple supported calls.

  • user.getLovedTracks
  • user.getRecentTracks
  • user.getBannedTracks

Support for more calls will be added in the future, but let's see how we can use these calls:

<?php $lastfm->getLoved(); ?>

The above call is to get the users loved tracks. Pretty easy. Let's look at the other calls.

Get recently played:

<?php $lastfm->getRecent(); ?>

Get banned tracks:

<?php $lastfm->getBanned(); ?>
